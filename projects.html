Projects

Event Guest Seating Planner (Web Tool)

Role: Designer & Developer

Goal: Create a simple planning tool to organize guest seating for an event in a clear, efficient way.

I designed and built a small web application using Bootstrap 4 and JavaScript to help plan where guests will sit at different tables. I focused on making the layout easy to read and updating the information in a way that reduces confusion and double-checking. The project simulates a real business scenario where someone needs to coordinate people, space, and logistics so the event runs smoothly.

Management-related skills demonstrated: planning and organizing information, thinking about user needs, attention to detail, and using technology to support event and operations management.

Customer Transaction & Probability Simulation (Dice Project)

Role: Developer

Goal: Use a probability simulation to understand random outcomes and practice logical thinking similar to basic business scenarios.

In this project, I built an interactive dice-rolling web page with JavaScript, HTML, and CSS. Teen Program Coordination – Skyline Library Internship

Role: Program Assistant & Event Coordinator

Goal: Support the planning, organization, and delivery of weekly teen programs while maintaining smooth library operations.

I helped coordinate weekly teen programs serving 20–40 participants. I assisted with planning activities, preparing materials, managing setup and cleanup, and interacting with teens during sessions. I also handled administrative tasks such as inventory, documentation, and attendance tracking to keep the program organized.

This experience gave me hands-on practice in scheduling, time management, coordinating resources, and ensuring activities ran smoothly. I learned to communicate with diverse groups, adapt quickly when plans changed, and maintain a positive environment even during busy program hours.

Management-related skills demonstrated: event planning, coordination, communication, task organization, operations support, and leadership.
